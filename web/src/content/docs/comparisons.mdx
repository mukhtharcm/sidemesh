Sidemesh exposes several ways to interact with the host and the agent. These pages help you choose the right one for each situation.

## Topics

- [When to Use What](/comparisons/when-to-use-what/) — quick decision guide

## Philosophy

Sidemesh is a control plane, not a replacement for your existing tools. The daemon exposes host and provider capabilities; the app gates UI based on what is available. When multiple options solve the same problem, we document the trade-offs so you can pick the right one.
