Welcome to the Sidemesh documentation. Sidemesh is a fleet-first mobile control plane for coding agent sessions. Run the daemon on your own machines and control everything from your phone or desktop.

## Getting started

<CardGrid stagger>
	<Card title="Install" icon="rocket">
		Get the daemon running on macOS, Linux, or Windows with WSL. [Install →](/getting-started/install/)
	</Card>
	<Card title="Connect" icon="iphone">
		Pair the mobile app with your host using a QR code or manual entry. [Connect →](/getting-started/connect/)
	</Card>
	<Card title="Daily workflow" icon="list-format">
		Start sessions, chat with agents, review approvals, and manage files. [Workflow →](/getting-started/workflow/)
	</Card>
	<Card title="Troubleshooting" icon="warning">
		Fix common connection, provider, and sync issues. [Troubleshooting →](/getting-started/troubleshooting/)
	</Card>
</CardGrid>
