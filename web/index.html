What it does

One calm surface for every peer in your fleet.

Register a machine once, reach it forever. Sidemesh keeps each host alive in its own lane so you can hop between a MacBook, a VPS, and a lab box without ceremony.

⌁

Multi-host, side-by-side

Add any machine with a label, a URL, and a shared token. Switch between them without pairing servers or cloud brokers.

>_

Sessions that persist

Start a Codex session on your desk, check in from the couch, resume on the train. Transcripts and live activity follow you.

✓

Approvals in your pocket

Risky commands and edits are surfaced as approvals you can grant or decline without opening a laptop.

ƒ

Workspace files

Browse, read, and edit files inside each session's workspace. No scp, no copy-paste, no detours.

~

Live activity

A streaming feed of what each agent is doing — commands, file writes, status — so nothing happens in the dark.

◊

Private by default

Runs over your own network. Works great over Tailscale. No third-party relay, no account to create.

How it works

A tiny daemon per machine. One app for all of them.

Each machine runs a small Node daemon that owns a local codex app-server. The Sidemesh app connects to each daemon over your network — no pairing server, no middleman.
