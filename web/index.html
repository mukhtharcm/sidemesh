Not a toy chat wrapper. Sidemesh tracks the things that matter when an agent is actually changing a codebase.

Multi-host control

Install path.

This is still private preview software. Use it only on machines you control.

1

Install the daemon

Use the private GitHub package on each trusted host.

npm install -g sidemesh-private-package
2

Configure providers

Pick Codex, Copilot, port, token, and optional terminal access.

sidemesh setup
sidemesh doctor
3

Pair the app

Start the daemon as a service, then add the host from your app.

sidemesh service install
sidemesh pair

Security posture

Private network first.

Sidemesh exposes powerful host abilities: agent chat, filesystem reads, approval actions, and optional terminals. Run it behind Tailscale or a trusted LAN, protect the bearer token, and avoid exposing it directly to the public internet.
